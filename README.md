# Hooke's Hike - a typescript platformer game using pixijs and matter
Climb through the coil of my monitor and break free using nothing but a spring.

### Deployment

To deploy this project run:

```bash
    npm run build
```

Or, to run locally:

```bash
    npm run dev
```
